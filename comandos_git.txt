git init . 		-> Cria um repo git na pasta atual
git status 		-> Mostra o status do trabalho
git add 		-> Adiciona arquivo para ser comitado
git commit -m 		-> Faz commit de arquivo que estava para ser commitado
git log 		-> Mostra o historico de commit

## FLUXO DE TRABALHO

1. criar o diretório do projeto
2. iniciar o gir dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome_arquivo>
6. git status
7. git commit -m "mensagem"
8. git status
9. Volta para o passo 3
